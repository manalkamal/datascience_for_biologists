Part 2: Identifying variable types from datasets


This exercise uses an example dataset, the first 10 rows of which are shown below which provides some information about all elements in the periodic table. It contains the following columns, which represent the variables in the dataset (don’t worry about units):

  • element_name: The full name of the element
  • acid_base: Whether the element is considered an acid or a base.
  • atomic_mass: The element’s mass
  • number_of_protons: The number of protons the element has
  • periodic_table_period: The period (row) in the periodic table where this element is found
  • periodic_table_group: The group (column) in the periodic table where this element is found
  • atomic_radius: The size of element’s atomic radius
  • electronegativity: A measure of an element’s propensity to attract electrons
  • metal_type: The type of metal a given element is (there is no natural ordering to this quantity)
  • density: The density of an element at room temperature.
  • specific_heat: The specific heat of an element.
